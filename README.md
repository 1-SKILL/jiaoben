# 脚本 (jiaoben)

> 短视频脚本工坊 · 5 分钟出完整脚本，产能提升 5 倍，完播率翻倍。

**jiaoben** 解决的不是"写什么"，是"怎么说才有人听"。Hook 公式库、平台完播率优化结构、口播节奏校准，每 2-3 天更新验证有效的新公式。

## Install

```bash
npx skills add 1-SKILL/jiaoben -g -y
```

## 解决什么问题

- 对着镜头不知道说什么，大脑一片空白
- 好不容易拍了，完播率 8%——前 3 秒没抓住人
- 一条脚本憋 2 小时，一周才能出 3 条

## 效果

| 指标 | 用前 | 用后 |
|------|-----|-----|
